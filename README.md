# Regex Fun

## Summary

Here we will be going over the regex used to verify emails `/^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,5})$/`. This type of expression allows for email adresses to be verified upon signing up etc.
